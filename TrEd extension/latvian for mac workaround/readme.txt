If standard input methods fail to give Latvian symbols, copy xmodmap.darwin given here to /Applications/tred/tredlib or /opt/tred/tredlib or /Users/_yourname_/tred/tredlib depending on where TrEd is installed. Make back-up copy of the original xmodmap.darwin beforehand!. To get Latvian symbols, hold right Alt.

Tested on iMac with default keyboard and OS X Yosemite 10.10.5
